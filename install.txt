PERSONAL ARCH INSTALL GUIDE

Refer to the Arch Installation Guide here:
https://wiki.archlinux.org/index.php/Installation_guide 

--SWAPFILE--
Do this after mounting the main partition but before pacstrap.
Create the swapfile with:
dd if=/dev/zero of=/mnt/swapfile bs=1M count=4096 status=progress
chmod 600 /mnt/swapfile
mkswap /mnt/swapfile
swapon /mnt/swapfile

--GRUB--
DOS
pacman -S grub
grub-install /dev/sdX
grub-mkconfig -o /boot/grub/grub.cfg

UEFI
pacman -S grub efibootmgr dosfstools
mkfs.fat -F32 /dev/sda1
mkdir /boot/EFI
mount /dev/sda1 /boot/EFI
grub-install --target=x86_64-efi --bootloader-id=grub_uefi --recheck
grub-mkconfig -o /boot/grub/grub.cfg

--USERS & SUDO--
Install sudo with:
pacman -S sudo

Create a user with:
useradd -m user_name
passwd user_name

Give user perms with:
usermod -aG wheel,audio,video user_name

Add user to sudoers with:
visudo (uncomment %wheel line)

--OTHER STUFF--
Bluetooth:
pacman -S bluez bluez-utils
systemctl start bluetooth

Wi-Fi:
pacman -S networkmanager
systemctl enable NetworkManager
nmtui

Ethernet:
pacman -S dhcpcd
systemctl enable dhcpcd

NVIDIA Drivers:
pacman -S nvidia nvidia-utils

--REBOOT TO MAKE SURE EVERYTHING IS WORKING--
exit
reboot

Install virtualbox guest utils:
sudo pacman -S virtualbox-guest-utils git

Clone archbox repo with:
mkdir ~/Downloads
cd ~/Downloads
git clone https://github.com/nateranda/archbox

--GNOME--
Install gnome with:
sudo pacman -S gnome xorg gnome-software-packagekit-plugin

(GDM won't be enabled by default in case something goes wrong.
Enable the GDM service with 'sudo systemctl enable gdm')

Install extensions & tweaks with:
sudo pacman -S gnome-shell-extensions gnome-tweaks

Command Line Way:
paru -S gnome-shell-extension-blur-my-shell-git gnome-shell-extension-audio-output-switcher-git gnome-shell-extension-arc-menu-git gnome-shell-extension-arch-update gnome-shell-extension-screenshot-git
then, restart GNOME

GUI Way (recommended):
Open firefox and install the GNOME Extension Firefox add-on.
Then, enable these extensions:
Blur My shell
Audio Output Switcher
Arch Update Indicator
Arc Menu
Screenshot Tool

Install Pop Shell with:
cd ~/Downloads
git clone https://github.com/pop-os/shell
cd shell
sudo pacman -S typescript
make
make install

Install a transparent GNOME terminal with:
paru -S gnome-terminal-transparency

Download the nord-darker theme with:
sudo pacman -S wget
mkdir ~/.themes
cd ~/.themes
wget https://github.com/EliverLara/Nordic/releases/download/v1.9.0/Nordic-darker.tar.xz
gsettings set org.gnome.desktop.interface gtk-theme "Nordic"
gsettings set org.gnome.desktop.wm.preferences theme "Nordic"

--QTILE--
